How to import the the Demo.postman_collection.json file as a collection in the Postman application:

1. In the Postman application in the upper left corner there is an "Import" button
2. Click that and you should see a new window open where you need to drag (or choose) the file
which you should have already downloaded (Demo.postman_collection.json).
3. When uploading is done you should be able to view all the requests from the collection.
